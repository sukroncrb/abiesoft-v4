# abiesoft-4.0
### Instalasi
1. Download file **abiesoft-v4.zip**, kemudian extract zip.
2. Rename file env_sample menjadi .env seting pengaturan yang dibutuhkan di env seperti konfigurasi database, apikey dan secretkey
3. mengupdate composer dengan perintah <code>composer update</code>
4. Mengimport database dengan <code>php abiesoft database:import</code>
5. Build binary golang dengan perintah <code>php abiesoft build</code>
6. Menjalankan aplikasi <code>php abiesoft start</code>
